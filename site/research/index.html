    Current Research at PAIC Lab

    Privacy-Preserving Machine Learning (PPML)

    • Research Focus: Developing machine learning techniques that preserve data privacy
    • Key Areas:
    • Homomorphic Encryption for secure computation
    • Federated Learning for distributed privacy-preserving training
    • Secure Multi-Party Computation (SMPC)
    • Model optimization and lightweight techniques

    Diffusion Model Optimization

    • Project: Lightweight Diffusion Models for Privacy-Preserving Applications
    • Goals:
    • Reduce computational complexity while maintaining model performance
    • Enable efficient deployment in privacy-sensitive environments
    • Optimize model architecture for encrypted computation
